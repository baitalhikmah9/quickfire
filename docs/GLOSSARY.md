# Glossary

## Promo Code

A redeemable code that grants wallet tokens after Convex validates its status, redemption caps, and account restrictions.

## Coupon Mode

An admin preset that defines who may redeem a promo code and how many times it can be redeemed.

## Restricted Account

The only user account allowed to redeem an account-scoped promo code.

## Wallet Transaction

An immutable token ledger entry created when tokens are granted, reserved, consumed, refunded, or adjusted.
